# Class总括
之前实例对象是通过构造函数，ES6 为了接近传统语言的写法，引入了 Class，作为对象的模板。 class 是语法糖，让对象原型的写法更加清晰、更像面向对象编程的语法而已。 

## 基本内容
创建:
1. class 本身就是函数，只是定义了一种构造函数的另外一种写法。 
2. 方法不需要用 function 关键字。
3. 方法间不需要逗号分隔。

使用：
1. new 命令。
2. 类的所有方法都定义在 prototype 上面。
